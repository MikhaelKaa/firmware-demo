# Unity Test Framework

Минимальная выдержка из [Unity](https://github.com/ThrowTheSwitch/Unity) v2.6.1 для юнит-тестирования embedded C-кода.

## Содержимое

Только ядро фреймворка (без примеров и документации):
- `src/unity.h` — публичный API
- `src/unity_internals.h` — внутренние определения
- `src/unity.c` — реализация
